# Sigora Docs

This directory consolidates the initial Sigora product and architecture notes into repository-local documentation.

## Documents

- [Architecture Design](./architecture.md): consolidated system architecture, trust boundaries, components, protocol, storage, security model, and v1 to v2 evolution
- [Implementation Strategy](./implementation-strategy.md): recommended Rust and Swift split, process model, repo layout, transport, and implementation order
- [Vision And Roadmap](./vision-and-roadmap.md): project positioning, AIR/AIP model, and the v1 to v2 path
- [V1 MVP Spec](./v1-mvp-spec.md): scope, requirements, data model, APIs, and acceptance criteria for the first usable release
- [V1 Implementation Notes](./v1-implementation-notes.md): suggested Rust workspace layout, runtime flow, CLI behavior, and macOS UI requirements

## Status

The current repository is still at the documentation and design stage. These docs are the working baseline for implementation.
